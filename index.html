No Monkey Business

We at CodeMonkey pride ourselves in responding to your viewing screen size. Having more fun than a barrel of monkeys. I'm the remote code monkey swinging fon a vine, dreaming of life on the other side. At night I swing on codeMonkey bars that help me forget about this monkey business. We code monkeys don't want a real business and have to wear all those hats. We just want to work on code so we can buy some more bananas.

This ain't no monkey-see-monkey-do site. This is Bootstrap 3.3.6 Responsive Web Design. We know our "poop", but we don't throw it out people. Not a chance. We're restrained even if not fully trained. We're always learning about our trade. You know the drill. Just when we start to get comfortable swinging on these trees somebody else creates a new flavor-of-the-week and suddenly we're out-of-date and irrelevant all over again.
